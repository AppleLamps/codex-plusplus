# codex-plusplus

Installer CLI for codex-plusplus, a local tweak system for Codex.app.

```sh
curl -fsSL https://raw.githubusercontent.com/AppleLamps/codex-plusplus/main/install.sh | bash
```

See the repository README for architecture, tweak authoring, security policy, and release notes:

https://github.com/AppleLamps/codex-plusplus
